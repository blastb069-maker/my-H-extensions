## Guide

Paste this URL into Aniyomi / Anikku (**Settings → Extensions → Extension repos → Add**):

```
https://raw.githubusercontent.com/blastb069-maker/my-H-extensions/repo/index.min.json
```

Notes:

- `repo` is the **branch name**, not a folder on `main`.
- The repo must include `repo.json` (generated by CI). Without it, the app shows “invalid URL”.
- Extensions are **debug** builds. If install fails, try **Settings → Advanced → Installer → Legacy** (or Private).

Built APKs are on the [`repo` branch](https://github.com/blastb069-maker/my-H-extensions/tree/repo/apk) after each successful CI run on `main`.
